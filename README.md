# Etch A Sketch

Interactive Etch-A-Sketch grid to make small pixel drawings. This project is part of the "Odin Project" curriculum.

*How to play:* You can access the sketch pad by opening the latest GitHub Pages deployment. You can color individual pixels in the grid by clicking them. If you want to make strokes, you can keep the left mouse button clicked and move the cursor over the pixels. By moving the slider that is labeled "grid size" you can adjust the grid. The maximum size is 64x64 pixels. If you want random rainbow pixel colors, click "activate rainbow mode". Clear all colored pixels using the "clear grid" button. Have fun creating little drawings!
